Инструменты

1) Карандаш (ctrl+p) - предназначен для рисования линий.

2) Рука (ctrl+h) - предназначена для пролистывания документа.

3) Прямоугольник (ctrl+s) - предназначен для рисования прямоугольников(shift - квадрат).

4) Треугольник - предназначен для рисования треугольников.

5) Эллипс (ctrl+j) - предназначен для рисования эллипсов(shift - круг).

6) Прямая (ctrl+b) - предназначен для рисования прямых линий.

7) Слой (ctrl+l) - предназначен для создания слоев.

8) Изображение (ctrl+i) - предназначен для вставки картинок(каждая картинка = новый слой).

9) Ластик (ctrl+g) - предназначен для стирания нарисованых элементов.

10) Пипетка (ctrl+n) - предназначена для получения цвета точки.

11) Заливка (ctrl+f) - предназначена заливки области.

12) Перемещение (ctrl+m) - предназначено для смещения слоев.

13) Ножницы (ctrl+u) - предназначены для обрезки слоев.

14) Текст (ctrl+q) - предназначен для рисования текста.

Слои

Справа находиться панель слоев, в ней находяться миниатюры слоев их названия и кнопки управления. Активный слой подсвечивается светло-синим цветом Cлой можно Удалить(клавиша delete удаляет активный слой или кнопка с мусорным ведром) Для востановления последнего удаленного слоя нужно нажать ctrl+e;

Также слои можно двигать

Каждый слой имеет историю изменений для возврата назад или вперед используйте стрелки справа вверху или ctrl+z, ctrl+y

Цвета

Сверху слева находиться панель цветов. Также можно выбрать любой цвет. Для этого нужно нажать на большой квадрат цвета.
